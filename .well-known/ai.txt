# AI Training
# Opt-in status: ALLOWED_WITH_ATTRIBUTION
# This code MAY be used for AI training with the following conditions:

## Conditions for AI Training
1. Attribution required: Credit must be given to "Accessibility Everywhere"
2. License preservation: Any derived works must preserve license terms
3. Ethical use: Training data must not be used to create inaccessible software
4. No adversarial use: May not be used to circumvent accessibility requirements
5. Accessibility-first: AI systems trained on this data should promote, not hinder, accessibility

## Specific Permissions
- ✅ Code completion/suggestion tools (GitHub Copilot, Tabnine, etc.)
- ✅ Documentation generation
- ✅ Accessibility testing automation
- ✅ Educational AI assistants
- ✅ Research into accessible software development

## Specific Restrictions
- ❌ Creating tools that bypass accessibility requirements
- ❌ Generating inaccessible code
- ❌ Training proprietary models without attribution
- ❌ Use in systems that discriminate against people with disabilities

## Attribution Format
When using this codebase for training:
```
This model was trained on code from Accessibility Everywhere
(https://github.com/accessibility-everywhere/accessibility-everywhere)
Licensed under MIT + Palimpsest v0.8
```
